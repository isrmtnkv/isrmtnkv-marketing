# Маркетолог на аутсорсе

Статический сайт для GitHub Pages.

## Локальный просмотр

```bash
python3 -m http.server 8877
```

Откройте `http://127.0.0.1:8877/index.html`.

## GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите файлы сайта.
3. В настройках репозитория откройте `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`.
6. Папка: `/root`.

После включения GitHub Pages сайт будет открываться сразу с главной страницы репозитория.
